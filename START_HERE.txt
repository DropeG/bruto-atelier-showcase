🚀 COMENZAR EN 5 MINUTOS
═══════════════════════════════════════════════════════════════════════════════

1. Clonar el repo:
   $ git clone https://github.com/DropeG/bruto-atelier-showcase.git
   $ cd bruto-atelier-showcase

2. Instalar dependencias:
   $ npm install

3. Iniciar servidor:
   $ npm run dev

4. Abrir en navegador:
   → http://localhost:8080

5. Ver en producción:
   → https://bruto-atelier-showcase.vercel.app

💻 INFORMACIÓN CRÍTICA
═══════════════════════════════════════════════════════════════════════════════

Tipo de app:
  ✓ React 18 SPA (Single Page Application)
  ✓ NO es Next.js
  ✓ Enrutamiento: React Router v6
  ✓ Build: Vite
  ✓ Deploy: Vercel (estático)

Propósito:
  ✓ Showcase/Portafolio de estudio de diseño
  ✓ Catálogo de muebles, arquitectura e interiorismo
  ✓ Soporte multimoneda (12 países)
  ✓ Carrusel de imágenes con efectos

Stack principal:
  ✓ React 18.3.1 - UI
  ✓ TypeScript 5.8.3 - Type safety
  ✓ Vite 5.4.19 - Build
  ✓ Tailwind CSS 3.4.17 - Styling
  ✓ Framer Motion 12.23.26 - Animaciones
  ✓ shadcn/ui + Radix UI - Componentes
  ✓ React Router 6.30.1 - Enrutamiento

Base de datos:
  ✗ NO hay base de datos
  → Todo es datos estáticos en /src/data/Gallery.tsx
  → Autenticación es fake (simulada)
  → Monedas son conversiones estáticas

Performance:
  ✓ Lazy loading de imágenes
  ✓ Blur backgrounds
  ✓ Caché en Vercel (1 año)
  ✓ Bundle: 481KB gzip (muy bueno)


🗂️ ESTRUCTURA RÁPIDA
═══════════════════════════════════════════════════════════════════════════════

src/
  pages/         → 8 páginas SPA (Index, Arquitectura, Piezas, etc.)
  components/    → 20+ componentes reutilizables
  components/ui/ → 30+ componentes shadcn/ui
  contexts/      → 3 contextos globales (Auth, Currency, Scroll)
  data/          → Galería con 10 items (20 imágenes)
  lib/           → Utilidades (blur-placeholders.ts)

public/images/ → 20+ carpetas de imágenes

Configuración:
  vite.config.ts       → Build tool
  tailwind.config.ts   → Estilos
  vercel.json         → Caché Vercel


🛣️ RUTAS PRINCIPALES
═══════════════════════════════════════════════════════════════════════════════

/                           → Home (10 imágenes + scroll snap)
/arquitectura              → Proyectos de arquitectura
/interiorismo              → Proyectos de interiorismo
/coleccion                 → Catálogo de colecciones
/serie                     → Series de productos
/piezas                    → Piezas con carrusel automático
/categoria/:cat/:col/:id   → Página dinámica
/auth                      → Login/Signup


🌐 CONTEXTOS GLOBALES
═══════════════════════════════════════════════════════════════════════════════

1. AuthContext
   → user, login(), logout(), signup()
   → Uso: const { user } = useAuth()

2. CurrencyContext
   → 12 países, conversión de monedas
   → Uso: const { selectedCountry } = useCurrency()

3. ScrollContext
   → Guarda posición al navegar
   → Uso: const { saveSectionId } = useScroll()

🐛 TROUBLESHOOTING RÁPIDO
═══════════════════════════════════════════════════════════════════════════════

Build falla:
  → npm install && npm run build

Dev no inicia:
  → npm install
  → Matar puerto 8080

ESLint errors:
  → npm run lint -- --fix

Imágenes no cargan:
  → Verificar /public/images/
  → DevTools > Network
